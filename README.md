# Mini Project with SIC

Repository ini berisi kumpulan tugas dan mini project yang saya kerjakan selama mengikuti program SIC (Samsung Innovation Campus).

## Deskripsi
Seluruh file dalam repository ini merupakan hasil latihan, tugas, dan mini project yang dikerjakan selama proses pembelajaran di SIC, khususnya pada materi dasar pemrograman dan pengolahan data menggunakan Python.

## Isi Repository
Beberapa jenis tugas yang terdapat dalam repository ini antara lain:
- Latihan dasar Python
- Tugas list dan dictionary
- Mini project konversi suhu
- Perhitungan BMI
- Latihan menggunakan Jupyter Notebook (.ipynb)

## Tujuan
- Mendokumentasikan hasil belajar selama mengikuti SIC
- Menjadi arsip tugas dan mini project
- Menunjukkan progres pembelajaran Python

## Catatan
- File ditulis dalam format Python (.py) dan Jupyter Notebook (.ipynb)
- Kode dibuat untuk keperluan pembelajaran
- Repository ini bersifat dokumentasi hasil belajar

Bahasa pemrograman: Python  
Program: Samsung Innovation Campus (SIC)
